# DelayFreq

DelayFreq is a precision delay-time and frequency reference tool for macOS, designed for producers who rely on accurate timing and tuning in every session.

## Support
If you need help, have found a bug, or want to request a feature:

• Create an Issue on this GitHub repository  
• Or email: delayfreq-support@proton.me

We're committed to keeping DelayFreq reliable and improving it with each update.
